## 💻 CBbot 是什么？

CBbot 是一个 **AI 原生智能体操作系统**，专门用于编程和任务自动化，支持本地和云端无缝集成。

**核心功能：**
- ✅ **SOTA AI 编程智能体**：内置强大的 AI 编程能力，支持全球最优秀的模型
- ✅ **零依赖安装**：无需 Docker、WSL 或任何外部工具 - 完全原生
- ✅ **一键登录**：点击即可打开浏览器登录 - 无需手动复制密钥
- ✅ **自动配置**：无需手动编辑配置文件
- ✅ **安全执行环境**：安全的运行时隔离保护您的系统免受未授权访问
- ✅ **可扩展技能系统**：预装技能库，兼容开源领域技能

## 💻 系统要求

> **🎯 零依赖**：无需 Docker、WSL 或任何外部工具 - 下载即用！

### macOS
- **操作系统**：macOS 10.15 (Catalina) 或更高版本
- **芯片**：仅支持 Apple Silicon (M1/M2/M3/M4) - 已在所有 M 系列芯片上测试
- **内存**：建议 8GB 或以上
- **磁盘空间**：至少 2GB 可用空间
- **依赖**：✅ **无** - 完全原生应用
- **网络**：AI 功能需要网络连接

### Windows
- **操作系统**：Windows 10 (64位) 或 Windows 11 必需
- **架构**：x64 (Intel/AMD 64位)
- **内存**：建议 8GB 或以上
- **磁盘空间**：至少 2GB 可用空间
- **依赖**：✅ **无** - 完全原生应用（无需 WSL）
- **网络**：AI 功能需要网络连接

## 📦 安装

### macOS 安装

1. **下载**：从 [Releases 页面](https://github.com/mobvoi/CBbot/releases/latest)下载最新的 `.dmg` 文件
2. **安装**： 
   - 双击 `CBbot.dmg`
   - 将 **CBbot** 拖到 **Applications** 文件夹
3. **启动与登录**： 
   - 打开 **Applications** → **CBbot**
   - 首次启动：安全对话框提示时点击 "打开"
   - 点击 **"登录"** 按钮 - 浏览器将自动打开
   - 在浏览器中完成登录 - CBbot 将自动检测并启动
   - 完成！无需复制密钥，无需手动配置

### Windows 安装

1. **下载**：从 [Releases 页面](https://github.com/mobvoi/CBbot/releases/latest)下载最新的 `.exe` 文件
2. **安装**： 
   - 运行 `CBbot-Setup.exe`
   - 按照安装向导操作
   - 选择安装目录（推荐默认）
3. **启动与登录**：
   - 在开始菜单中找到 **CBbot**
   - 点击 **"登录"** 按钮 - 浏览器将自动打开
   - 在浏览器中完成登录 - CBbot 将自动检测并启动
   - 完成！无需复制密钥，无需手动配置

> **💡 轻松登录**：CBbot 使用浏览器登录 - 只需点击按钮并登录。应用会自动检测您的登录状态并启动。无需手动复制任何密钥或令牌！

## ⚙️ 高级配置（可选）

### 配置飞书机器人

启用飞书机器人功能：

1. 打开 CBbot 应用
2. 进入 **"设置" → "机器人配置" → "飞书"**
3. 输入您的 **App ID** 和 **App Secret**
4. 点击 **"保存"** 并重启服务

### 配置 Telegram 机器人

启用 Telegram 机器人功能：

1. 打开 CBbot 应用
2. 进入 **"设置" → "机器人配置" → "Telegram"**
3. 输入您的 **Bot Token** 和 **Username**
4. 点击 **"保存"** 并重启服务

### 配置 Discord 机器人

启用 Discord 机器人功能：

1. 打开 CBbot 应用
2. 进入 **"设置" → "机器人配置" → "Discord"**
3. 输入您的 **Bot Token**
4. 点击 **"保存"** 并重启服务

> **💡 提示：** 所有配置都通过 CBbot 图形界面完成 - 无需手动编辑配置文件！

## 🛠️ 故障排除

### 常见问题

**CBbot 无法启动：**
- 确保有稳定的互联网连接
- 检查系统符合最低要求（8GB RAM，2GB 磁盘空间）
- 重启计算机后再试
- Windows：首次启动以管理员身份运行

**无法访问界面：**
- 首次启动等待 30-60 秒
- 检查防火墙是否阻止 CBbot
- 尝试重启应用程序

**密钥绑定失败：**
- 验证您的密钥没有多余空格
- 检查网络连接
- 从 [CodeBanana](https://www.codebanana.com) 获取新密钥

### macOS 专属

**"无法打开"安全警告：**
- 进入 **系统设置** → **隐私与安全性**
- 点击 CBbot 警告旁的 **仍要打开**
- 在确认对话框中点击 **打开**

**权限被拒绝错误：**
- 在提示时授予所需权限
- 检查 **系统设置** → **隐私与安全性** → **文件和文件夹**

### Windows 专属

**"Windows 已保护你的电脑"警告：**
- 点击 **更多信息**
- 点击 **仍要运行**
- 这对新应用程序是正常的

**防火墙阻止：**
- 将 CBbot 添加到 Windows 防火墙例外
- 进入 **Windows 安全中心** → **防火墙和网络保护** → **允许应用**

**安装失败：**
- 以管理员身份运行安装程序（右键 → "以管理员身份运行"）
- 安装期间临时禁用杀毒软件
- 安装到默认目录（避免包含空格或特殊字符的路径）
